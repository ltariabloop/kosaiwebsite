# KOSAI Wellness and Consulting Website

Official website for KOSAI Wellness and Consulting.

## Project Structure

- `kosai-official-website/` - Main website files (HTML, CSS, JS, images)
- `scripts/` - Python scripts for maintenance and updates
- `docs/` - Documentation files

## Local Development

To run the website locally:

```bash
cd kosai-official-website
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

## Deployment

This website can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service
