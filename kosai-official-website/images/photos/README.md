# Website Photos Directory

This directory contains all photos and images used throughout the KOSAI website.

## Structure

- `hero/` - Hero banner images
- `carousel/` - Carousel/slider images
- `team/` - Team member photos
- `projects/` - Project showcase images
- `logos/` - Logo files
- `icons/` - Icon files

## Current Image Locations

Images are currently referenced from various locations:
- `themes/custom/banyantree/images/` - Theme images
- `sites/btg/files/` - Site-specific files
- External URLs (Unsplash, etc.)

## Migration Plan

To organize images:
1. Copy all local images to this directory
2. Update image references in HTML files
3. Organize by category in subdirectories
